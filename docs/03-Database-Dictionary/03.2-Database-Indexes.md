# DATABASE INDEX STRATEGY

Version: 4.0

Status: APPROVED

---

# TUJUAN

Menjamin:

```text
Fast Lookup
Fast Reporting
Fast Dashboard
Fast RLS
```

---

# SISWAS

```sql
CREATE INDEX idx_siswa_nisn
ON siswas(nisn);
```

```sql
CREATE INDEX idx_siswa_nama
ON siswas(nama);
```

---

# GURUS

```sql
CREATE INDEX idx_guru_nip

```

```sql
CREATE INDEX idx_guru_nama

```


---

# JSONB INDEX

```sql
CREATE INDEX idx_rapor_jsonb
ON rapor_snapshots
USING gin(data_rapor);
```

---

```sql
CREATE INDEX idx_alumni_snapshot_jsonb
ON alumni_snapshots
USING gin(akademik_snapshot);
```

# RLS INDEXES

Mandatory:

```sql
guru_id
academic_term_id
kelas_id
```

harus memiliki index.
