# TUJUAN

Menghilangkan ketergantungan pada frontend.

---

# UPDATED_AT TRIGGER

Semua tabel transaksi.

```sql
CREATE FUNCTION update_timestamp()
```

---

Trigger:

```sql
BEFORE UPDATE
```

---

Update:

```sql
updated_at = now()
```

---

# AUDIT TRIGGER

Tabel:

```text
assessments
assessment_details
kehadiran
rapor_snapshots
kelas
pembagian_mengajar
```

---

Trigger:

```sql
AFTER INSERT
AFTER UPDATE
AFTER DELETE
```

---

Write to:

```sql
audit_logs
```

---
