# TUJUAN

Reporting Engine merupakan pusat agregasi data seluruh modul SIKAD.

Modul ini bertugas menghasilkan:

```text
Laporan Operasional
Laporan Akademik
Laporan Kurikulum
Laporan Beban Kerja
Laporan Kehadiran
Laporan Alumni
Dashboard Analytics
```

tanpa melakukan query langsung ke tabel transaksi mentah.

---

# FILOSOFI DESAIN

Dashboard tidak boleh membaca:

```text
assessment_details
kehadiran
pembagian_mengajar
```

secara langsung.

---

Semua laporan harus melalui:

```text
Analytics Layer
```
