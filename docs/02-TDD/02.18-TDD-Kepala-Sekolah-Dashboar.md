Version: 4.0

Status: APPROVED

# TUJUAN

Dashboard Kepala Sekolah merupakan pusat monitoring strategis seluruh aktivitas akademik sekolah.

Dashboard ini tidak digunakan untuk operasional harian.

Dashboard digunakan untuk:

```text
Monitoring
Evaluasi
Pengambilan Keputusan
Pengawasan
Perencanaan
```

---

# FILOSOFI DESAIN

Kepala Sekolah tidak membutuhkan:

```text
Input Nilai
Input Kehadiran
CRUD Data
```

---

Kepala Sekolah membutuhkan:

```text
Insight
Trend
Warning
KPI
Ringkasan
```

---

# EXECUTIVE OVERVIEW

Saat login, Kepala Sekolah langsung melihat:

```text

Jumlah Siswa
Jumlah Kelas
Jumlah Mapel
Persentase Kehadiran
Rata-Rata Nilai
Distribusi Beban Kerja
Status Semester
```

# KPI ENGINE

## KPI Akademik

Formula:

```text
Total Nilai
÷
Jumlah Nilai
```

---

Output:

```text

```

---

## KPI Kehadiran

Formula:

```text
Hadir
÷
Total Kehadiran
```

---

Output:

```text

```

---

## KPI Guru

